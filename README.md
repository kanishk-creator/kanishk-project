# Kanishk's website

Clone this repo

```bash
# for gh users
gh repo clone whoisguy/kanishk-project

# for git users
git clone https://github.com/whoisguy/kanishk-project.git
```

Start development server

```bash
# for npm users
npm dev

# for yarn users
yarn dev

# for pnpm users (recommended)
pnpm dev
```
